How to set default volume for a device:

- `acpi_listen` and plug/unplug the device
	- let's say we are interested in `jack/headphone HEADPHONE plug`
- write a script in `/etc/acpi/`
	- example: `echo pacmd set-source-volume alsa_input.pci-0000_00_1f.3-platform-skl_hda_dsp_generic.HiFi__hw_sofhdadsp__source 0x8000 > /etc/acpi/headphone-set-mic-vol.sh`
	- `pacmd list-sources` might help
- put a file in `/etc/acpi/events/`
	`printf 'event=jack/headphone HEADPHONE plug\naction=/etc/acpi/headphone-set-mic-vol.sh\n' > /etc/acpi/events/headphone-plugged`
